# TrackIt

An application for recording and tracking personal financial transactions

## About

In this app you can register a new account, login into your account and register any transaction you've done.

All data are stored in a public API.

I created this project with the aim of creating a usefull tool to help people with their finances and keep their monetary transactions tracked in addition to challenging myself.

## How to run

You can just access the deploy in: https://projeto13-mywallet-front-two.vercel.app/

or

You can clone the repository throught:
```
git clone https://github.com/Masih-Saldanha/projeto13-mywallet-front.git
cd projeto13-mywallet-front
```
Then you just need to install the dependecies throught:
```
npm i
```
And finally you can run the application with the following command:
```
npm start
```
The application will open on your localhost in the port 3000.
